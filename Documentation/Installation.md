
# Installation

<br>

## Steps

1. Download a **[Release]** of your choice.

2. Open **MBlock**.

3. Drag & drop the downloaded file into the editor.

4. Wait for the confirmation message.

5. Close & reopen **MBlock**.

6. Navigate to a sprite, open the extension menu & add DeltaBlock.
