Arduino

Mode

Sets the IO mode of a pin.

State

Sets the state of the output pin.

To Float

Casts the input to a float value.

Var Declare

Declares a variable with the chosen type
at the position the block was called at.

Var Set

Sets a variable to the input.
(Converts the input based on the variable type)

Var Assign

Assigment operators.

Var Get

Returns the value of the variable.

From To

Loops from the [from] value to the [to] value
while making the current index available via the [index] variable.
