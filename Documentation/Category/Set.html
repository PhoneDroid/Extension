Set

Clear All

Clears all existing sets.

Clear

Clears the specified set.

New

Creates a new set with the specified name.
(Name can only contain unicode letters, _, -, 0-9)

New With

Does the same as New, while adding the given values.
(Values are seperated with a ,)
(keys are enclosed in []:)

Add

Adds a value to the set.

Remove

Removes the specified values from the set.

Size

Returns the size of the set.

Contains

Returns whether or not the set contains the specified value.

Value At

Returns the value at the specified index.
