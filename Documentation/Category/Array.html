Array

Clear All

Clears all existing arrays.

New

Creates a new array with the specified name.
(Name can only contain unicode letters, _, -, 0-9)

Add

Pushes a value onto the end of the array.

Pop

Removes and returns the last value of the array.

Shift

Removes and returns the first value of the array.

Unshift

Adds the value to the start of the array.

Value At

Returns the value at the specified index.

Length

Returns the length of the array.

Concat

The first and second arrays are combined into the third.
