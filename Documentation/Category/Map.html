Map

Clear All

Clears all existing maps.

Clear

Clears the specified map.

New

Creates a new map with the specified name.
(Name can only contain unicode letters, _, -, 0-9)

New With

Does the same as New, while adding the given key-value pairs.
(key-value pairs are seperated with a ,)
(keys are enclosed in []:)

Put

Puts a value with the specified key into the map.

Delete

Deletes the value under the specified key.

Size

Returns the size of the map.

Has

Returns whether or not the map contains the specified key.

Get

Returns the value at the specified key.
